# DermaClust Skincare Website

DermaClust is a responsive skincare product recommendation website built using **HTML, CSS, and JavaScript**. The website provides an interactive user interface for browsing skincare products based on different skin concerns such as acne, acne marks, pores, and pigmentation.

## Features

- Responsive design for desktop, tablet, and mobile devices
- Interactive hero banner with auto-scrolling image slider
- Product sections categorized by skin concerns
- Dynamic popup alerts using JavaScript
- Clean and modern skincare-themed UI
- Navigation between home and product description pages

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons

## Dynamic Functionalities

### Hero Slider
The hero banner automatically scrolls through promotional images every 3 seconds using JavaScript.

### Interactive Alerts
When users click on:
- Acne
- Acne Marks
- Pores
- Pigmentation
- Cart / Under-development sections

A popup message appears:

```text
We are working on this page!
```

## Setup Instructions
